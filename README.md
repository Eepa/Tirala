Tirala

======


Tiralabra


Huffman koodauksella toteutettu pakkausohjelma. 

Jar-tiedoston voi ajaa Windowsissa käskyllä .jar-tiedoston 
sisältävästä kansiosta "java -jar HuffmanCoding.jar" 
(ääkköset eivät toimi tällöin oikein).

Linuxisssa käskyllä "java -jar -Dfile.encoding=UTF-8 HuffmanCoding.jar" 
saa ääkkösetkin toimimaan.

Ohjelma kysyy ensiksi, halutaanko tiedostoja pakata vai purkaa. 
Sen jälkeen ohjelma pyytää antamaan pakattavan tai purettavan tiedoston tiedostopolun. 
Ohjelma ilmoittaa onnistuneesta pakkaamisesta tai purkamisesta tai mahdollisista virhetilanteista.

Esimerkkisyöte pakkaukseen voisi olla (oletetaan että testi.txt -tiedosto on olemassa): 
”pakkaus[enter]HuffmanCoding\src\testi.txt[enter]”. 

Esimerkkisyöte purkamiseen voisi olla (oletetaan että pakattu tiedosto pakattutesti.txt.ep on olemassa): 
”purku[enter]HuffmanCoding\src\pakattutesti.txt.ep[enter]”.
